## 📌 What Is This Project?

**NeuroSense.AI** is a portable **EEG monitoring prototype** built on Arduino that captures brain electrical activity and displays it as a real-time waveform.

Instead of relying on:

- expensive, hospital-only EEG machines
- bulky clinical equipment
- trained neurologists for basic monitoring

NeuroSense builds a **low-cost embedded system** that acquires, processes, and visualizes brain signals for education and preliminary observation.

> ⚠️ Intended for educational purposes and preliminary monitoring only — **not** for clinical diagnosis.

---

## ⚠️ The Problem

Neurological disorders such as epilepsy require continuous monitoring of brain activity, but existing EEG systems have major limitations:

- Expensive and bulky hardware
- Generally available only in hospitals
- Require trained neurologists to interpret results
- Difficult to use for early or continuous monitoring

**The goal:** build a low-cost, portable, real-time EEG system that acquires brain signals, displays them, and helps identify abnormal activity patterns for preliminary monitoring.

---

## ✅ The Solution

- EEG **electrodes** on the scalp capture weak brain electrical signals
- An **EEG sensor module** amplifies and filters the signal
- **Arduino UNO** reads the signal and converts it to a digital value
- The digital values are sent to a computer via **USB serial communication**
- The **Arduino Serial Plotter** displays the signal as a live waveform

## ⚙️ Working Flow — Step by Step

1. **Wear EEG Electrodes** — placed on the scalp using a headband and conductive gel to capture electrical activity
2. **Brain Signal Acquisition** — electrodes detect tiny electrical signals generated by brain activity
3. **EEG Sensor Module** — amplifies and filters the weak signals, reducing noise
4. **Arduino Processing** — reads the analog signal on pin `A0` using `analogRead()`, converting it to a digital value (0–1023)
5. **Serial Communication** — sends digital values to the computer via USB using `Serial.println()`
6. **Visualization** — Arduino Serial Plotter displays incoming values as a real-time waveform
7. **Observation** — waveform is watched for spikes, fluctuations, or irregular patterns

---

## 📥 Input

The input is the brain's electrical activity (EEG signal):

```
Brain Activity → EEG Electrodes → EEG Sensor Module → Arduino A0 Pin
```

The Arduino does not receive words or images — only an **analog electrical signal** from the EEG sensor.

---

## 💻 How Arduino Reads the Data

The Arduino continuously executes:

```cpp
int eegValue = analogRead(A0);
```

This reads the voltage from the EEG sensor and converts it into a digital value.

**Example readings over time:**
| Reading | Value |
|---|---|
| 1 | 420 |
| 2 | 510 |
| 3 | 620 |
| 4 | 710 |
| 5 | 850 |
| 6 | 930 |

---

## 📤 Output

The Arduino sends each value to the computer:

```cpp
Serial.println(eegValue);
```

The Serial Plotter then displays these values as a moving waveform:

```
Brain Signal
    /\/\___/\_/\/\____/\_
```

If the program includes thresholds, it may also print demonstration messages such as:

- `Blink Detected`
- `High Signal Activity`

> These are demonstration labels based on defined thresholds — **not medical diagnoses**.
